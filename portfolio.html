Software Development Portfolio

Solar System Project

This was a pair programming project where we created an API that connects with a postgres database to store, retrieve, update, and delete planet information. This was developed in Python using Flask. It is deployed via Heroku.

Task List API Project

This was a solo project where I created a RESTful API to manage a users' tasks and goals. I wrote different routes that handled retreiving, deleting, creating, and updating said tasks and goals. This uses a postgres database and is deployed via Heroku.

Travel Map Project

This is a personal project I created based off a tutorial using the Python folium library. User input for a place name is entered in the terminal and the coordiantes are retrived by calling the LocationIQ API. Multiple places can be entered and when the user is done, the webpage can be opened to view all entered locations on a map.

Viewing Party Project

This was a solo project where a complex data structure is used to store movie information. Movies can be retrieved by who watched them, who needs a recommendation, by genre, favorite genre, and more.
